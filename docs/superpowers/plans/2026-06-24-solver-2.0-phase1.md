# Bestemshe Solver 2.0 — Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Raise solve throughput (target ≈20 M states/s, from ≈5.8 M) by removing the per-write CAS, load-balancing the sweep, and ordering moves for earlier cutoffs — with zero change to on-disk output.

**Architecture:** Introduce a templated per-state value store (`ByteStore` = 1 byte/state, CAS-free relaxed access; `PackedStore` = today's 2-bit CAS path). The sweep becomes a template `solve_impl<Store>` dispatched once by `choose_store_mode()`. The sweep loop switches from manual static chunking to dynamic 64 K-state blocks (each block decodes the board once via `UnindexState`, then O(1) odometer steps). Moves are reordered so opponent-side-landing pits (capture-likely) are tried first. A shell regression harness proves every step is byte-identical to a captured golden.

**Tech Stack:** C++17, OpenMP (`g++-16`), ZSTD/LZ4 (already linked). No new dependencies.

## Global Constraints

- **Bracketless style:** omit braces on single simple-statement bodies; keep them for compound bodies, `if/else` chains with any compound branch, and OpenMP structured blocks. (See [[bestemshe-code-style]].)
- **Build command (exact):** `g++-16 -O3 -std=c++17 -fopenmp -I/opt/homebrew/include -L/opt/homebrew/lib main.cpp Solver.cpp Inference.cpp Splitter.cpp Compressor.cpp -llz4 -lzstd -o bestemshe`
- **On-disk output is invariant:** every task must produce win/draw monoliths byte-identical to `tests/golden/`. The regression harness is the gate.
- **Block size:** `BLOCK = 65536` states. **RAM budget:** `RAM_BUDGET = 14ull << 30`.
- **Phase 2 is OUT OF SCOPE:** no pair-by-pair, selective preload, or `NibbleStore` here (the `StoreMode::NIBBLE` enumerator may be declared but is unused in Phase 1).
- Run the harness only when no production `--solve` is active (it writes/removes `layers/layer{M}_*.bin`).

```cpp
#pragma once
#include <vector>
#include <atomic>
#include <cstdint>
#include "Inference.h"   // GameValue

namespace Bestemshe {

enum class StoreMode { ONE_BYTE, TWO_BIT, NIBBLE };   // NIBBLE reserved for Phase 2

// 4 states per byte (2 bits each). Writes use a CAS loop because 4 states share a byte.
struct PackedStore {
    std::vector<std::atomic<uint8_t>> v;
    explicit PackedStore(uint64_t n) : v((n + 3) / 4) {
        #pragma omp parallel for schedule(static)
        for (uint64_t i = 0; i < v.size(); ++i) v[i].store(0, std::memory_order_relaxed);
    }
    inline GameValue read(uint64_t i) const {
        uint8_t byte = v[i / 4].load(std::memory_order_acquire);
        return static_cast<GameValue>((byte >> ((i % 4) * 2)) & 0x03);
    }
    inline void write(uint64_t i, GameValue g) {
        uint64_t b = i / 4;
        uint8_t shift = static_cast<uint8_t>((i % 4) * 2);
        uint8_t mask = static_cast<uint8_t>(~(0x03u << shift));
        uint8_t bits = static_cast<uint8_t>((static_cast<uint8_t>(g) & 0x03u) << shift);
        uint8_t cur = v[b].load(std::memory_order_acquire);
        while (!v[b].compare_exchange_weak(cur, static_cast<uint8_t>((cur & mask) | bits),
               std::memory_order_release, std::memory_order_acquire));
    }
};

// 1 atomic byte per state. CAS-free: relaxed store/load. 4x the memory of PackedStore.
struct ByteStore {
    std::vector<std::atomic<uint8_t>> v;
    explicit ByteStore(uint64_t n) : v(n) {
        #pragma omp parallel for schedule(static)
        for (uint64_t i = 0; i < n; ++i) v[i].store(0, std::memory_order_relaxed);
    }
    inline GameValue read(uint64_t i) const {
        return static_cast<GameValue>(v[i].load(std::memory_order_relaxed));
    }
    inline void write(uint64_t i, GameValue g) {
        v[i].store(static_cast<uint8_t>(g), std::memory_order_relaxed);
    }
};

} // namespace Bestemshe
```

template <class Store>
void RetrogradeSolver::solve_impl(Store& store) {
    inference_engine->preload_all_layers(layer_M);
    auto t_start = std::chrono::high_resolution_clock::now();
    const int R = 50 - static_cast<int>(layer_M);
    int iteration = 0;

    while (true) {
        iteration++;
        std::atomic<uint64_t> proven{0};

        #pragma omp parallel
        {
            uint64_t threads = static_cast<uint64_t>(omp_get_num_threads());
            uint64_t tid = static_cast<uint64_t>(omp_get_thread_num());
            uint64_t chunk = (num_states + threads - 1) / threads;
            uint64_t begin = tid * chunk;
            uint64_t end = std::min(num_states, begin + chunk);

            if (begin < end) {
                SolverState s = UnindexState(begin, layer_M);
                for (uint64_t i = begin; i < end; ++i) {
                    if (store.read(i) == GameValue::UNKNOWN) {
                        bool all_losses_for_me = true;
                        bool proved_win = false;
                        for (int move = 0; move < 5; ++move) {
                            if (s.board[move] == 0) continue;
                            SolverState next_s;
                            bool empties_opponent;
                            bool is_capture = execute_and_flip(s, move, next_s, empties_opponent);
                            GameValue tv;
                            if (empties_opponent || next_s.K_opp >= 26) tv = GameValue::LOSS;
                            else if (is_capture) tv = inference_engine->query_state(next_s.K_self + next_s.K_opp, next_s.K_opp, IndexState(next_s));
                            else tv = store.read(IndexState(next_s));
                            if (tv == GameValue::LOSS) { proved_win = true; break; }
                            if (tv == GameValue::DRAW || tv == GameValue::UNKNOWN) all_losses_for_me = false;
                        }
                        if (proved_win) { store.write(i, GameValue::WIN); proven.fetch_add(1, std::memory_order_relaxed); }
                        else if (all_losses_for_me) { store.write(i, GameValue::LOSS); proven.fetch_add(1, std::memory_order_relaxed); }
                    }
                    if (i + 1 < end) advance_solver_state(s, R);
                }
            }
        }

        std::cout << "  Iter " << iteration << ": Proven " << proven.load(std::memory_order_relaxed) << " states.\n";
        if (proven.load(std::memory_order_relaxed) == 0) break;
    }

    // Remaining UNKNOWN states form inescapable capture-free cycles -> DRAW.
    uint64_t draws = 0;
    #pragma omp parallel for schedule(dynamic, 65536) reduction(+:draws)
    for (uint64_t i = 0; i < num_states; ++i)
        if (store.read(i) == GameValue::UNKNOWN) { store.write(i, GameValue::DRAW); draws++; }

    auto t_end = std::chrono::high_resolution_clock::now();
    std::chrono::duration<double> d = t_end - t_start;
    std::cout << "[INFO] Detected " << draws << " loop-based Draws.\n";
    std::cout << "[BENCHMARK] Total Value Iteration: " << d.count() << "s\n";
}

template <class Store>
void RetrogradeSolver::pack_and_write(const Store& store, const std::string& out_dir) {
    std::filesystem::create_directories(out_dir);
    std::string win_file = out_dir + "/layer" + std::to_string(layer_M) + "_win.bin";
    std::string draw_file = out_dir + "/layer" + std::to_string(layer_M) + "_draw.bin";

    int k_count = StateIndex::GetKCount(layer_M);
    int R = 50 - layer_M;
    uint64_t b_count = StateIndex::nCr(R + 9, 9);
    uint64_t b_bytes = (b_count + 7) / 8;
    uint64_t total_bytes = static_cast<uint64_t>(k_count) * b_bytes;
    std::vector<uint8_t> win_bits(total_bytes, 0);
    std::vector<uint8_t> draw_bits(total_bytes, 0);

    #pragma omp parallel for
    for (int k_idx = 0; k_idx < k_count; ++k_idx) {
        uint64_t base_idx = k_idx * b_count;
        uint64_t byte_offset = k_idx * b_bytes;
        for (uint64_t j = 0; j < b_count; ++j) {
            uint8_t val = static_cast<uint8_t>(store.read(base_idx + j));
            if (val == static_cast<uint8_t>(GameValue::WIN)) win_bits[byte_offset + j / 8] |= (1 << (j % 8));
            else if (val == static_cast<uint8_t>(GameValue::DRAW)) draw_bits[byte_offset + j / 8] |= (1 << (j % 8));
        }
    }

    std::ofstream w_out(win_file, std::ios::binary);
    w_out.write(reinterpret_cast<const char*>(win_bits.data()), win_bits.size());
    std::ofstream d_out(draw_file, std::ios::binary);
    d_out.write(reinterpret_cast<const char*>(draw_bits.data()), draw_bits.size());
    std::cout << "[INFO] Saved monoliths to: " << win_file << " and " << draw_file << std::endl;
}

uint64_t RetrogradeSolver::estimate_preload_bytes(uint8_t M) const {
    uint64_t total = 0;
    for (int Mp = M + 2; Mp <= 48; Mp += 2) total += (StateIndex::GetLayerSize(static_cast<uint8_t>(Mp)) + 3) / 4;
    return total;
}

StoreMode RetrogradeSolver::choose_store_mode() const {
    // (env override is honored from Task 3; for now always TWO_BIT)
    return StoreMode::TWO_BIT;
}

void RetrogradeSolver::solve_layer_lock_free(const std::string& out_dir) {
    PackedStore store(num_states);
    solve_impl(store);
    pack_and_write(store, out_dir);
}
```

                        else if (is_capture) tv = inference_engine->query_state(next_s.K_self + next_s.K_opp, next_s.K_opp, IndexState(next_s));
                        else tv = read_local(IndexState(next_s));
                        if (tv == GameValue::LOSS) { proved_win = true; break; }
                        if (tv == GameValue::DRAW || tv == GameValue::UNKNOWN) all_losses_for_me = false;
                    }
                    if (!has_moves) { write_local(i, GameValue::LOSS); states_proven.fetch_add(1, std::memory_order_relaxed); }
                    else if (proved_win) { write_local(i, GameValue::WIN); states_proven.fetch_add(1, std::memory_order_relaxed); }
                    else if (all_losses_for_me) { write_local(i, GameValue::LOSS); states_proven.fetch_add(1, std::memory_order_relaxed); }
                }
                if (i + 1 < end) advance_solver_state(s, R);
            }
        }
```
