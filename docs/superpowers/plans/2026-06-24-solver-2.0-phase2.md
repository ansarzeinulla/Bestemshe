# Bestemshe Solver 2.0 — Phase 2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Solve a layer one symmetric kazan pair at a time, preloading only the higher micro-layers that pair can reach, so the descent to M=0 fits ~13 GB peak on a 16 GB host while producing micro-layer `.raw` files byte-identical to the whole-layer `--solve` + `--split` path.

**Architecture:** Layer M partitions into independent subgraphs `{(K1,K2),(K2,K1)}` (spec §3.1). `solve_layer_pairwise` loops the pairs; each pair allocates only its own `slices·b_count` store (1 or 2 micro-layers), value-iterates with local slice indexing, and writes its `.raw` planes directly. The InferenceEngine preloads exactly `Reach(K1,K2)` per pair and **evicts between pairs** to keep RAM bounded. Store type is chosen per pair by the adaptive budget using the *selective* preload size.

**Tech Stack:** C++17, OpenMP (`g++-16`), ZSTD/LZ4. Reuses Phase 1's `StateStore.h`, harness, and `RAM_BUDGET`.

## Global Constraints

- **DEPENDS ON Phase 1 plan Tasks 1–3** (`docs/superpowers/plans/2026-06-24-solver-2.0-phase1.md`): the regression harness (`tests/*.sh`, `tests/golden/`), `StateStore.h` (`ByteStore`, `PackedStore`, `StoreMode`), and `choose_store_mode` + `static constexpr uint64_t RAM_BUDGET = 14ull << 30;` in `Solver.cpp`. **Execute Phase 1 Tasks 1–3 before starting Phase 2.** Phase 1 Tasks 4–5 (dynamic odometer, move ordering) are optional and independent.
- **Bracketless style:** omit braces on single simple-statement bodies; keep them for compound bodies, `if/else` chains with any compound branch, and OpenMP structured blocks. (See [[bestemshe-code-style]].)
- **Build command (exact):** `g++-16 -O3 -std=c++17 -fopenmp -I/opt/homebrew/include -L/opt/homebrew/lib main.cpp Solver.cpp Inference.cpp Splitter.cpp Compressor.cpp -llz4 -lzstd -o bestemshe`
- **Output invariant:** pairwise `layer_K1_K2_{win,draw}.raw` must be byte-identical to whole-layer `--solve` + `--split` for the same layer. `tests/regression_pairwise.sh` is the gate.
- **RAM bound:** evict the InferenceEngine cache (`clear_cache()`) before each pair's `preload_pair`, so peak ≈ one pair's store + its selective preload. Real peak ≈ 13.3 GB at M=2 (tight; close other apps for that layer). `NibbleStore` is NOT used.
- **Block size:** `BLOCK = 65536`. **Store choice:** `1-byte if (pair_states + selective_preload_bytes) ≤ RAM_BUDGET, else 2-bit` (env override `BESTEMSHE_STORE_MODE=byte|packed`).
- Run harnesses only when no production solve is active.

```cpp
// Soundness check: every micro-layer (k1',k2') reachable by a capture from any state of
// pair {(K1,K2),(K2,K1)} must be in Reach(K1,K2) (the preload set). Small pairs only.
int RunReachTest(int K1, int K2) {
    int M = K1 + K2, R = 50 - M;
    uint64_t b_count = StateIndex::nCr(R + 9, 9);
    int min_K = StateIndex::GetMinK(static_cast<uint8_t>(M));

    std::set<std::pair<int,int>> formula;
    for (int j = K1 + 2; j <= 24; j += 2) formula.insert({K2, j});
    for (int j = K2 + 2; j <= 24; j += 2) formula.insert({K1, j});

    std::set<std::pair<int,int>> brute;
    int orient[2][2] = {{K1, K2}, {K2, K1}};
    int norient = (K1 == K2) ? 1 : 2;
    for (int o = 0; o < norient; ++o) {
        int Ks = orient[o][0];
        uint64_t I_K = (Ks - min_K) / 2;
        State s = StateIndex::UnindexState(I_K * b_count, static_cast<uint8_t>(M));

            for (int move = 0; move < 5; ++move) {
                if (s.board[move] == 0) continue;
                State f;
                bool emp;
                bool cap = ExecuteMoveAndFlip(s, move, f, emp);
                if (cap && !emp && f.K_opp < 26) brute.insert({f.K_self, f.K_opp});   // queried (k1',k2')
            }
            if (jb + 1 < b_count) StateIndex::AdvanceBoard(s.board);
        }
    }

    int bad = 0;
    for (auto& p : brute) if (!formula.count(p)) { std::cerr << "  UNCOVERED (" << p.first << "," << p.second << ")\n"; bad++; }
    if (bad == 0) std::cout << "[REACHTEST PASS] (" << K1 << "," << K2 << "): " << brute.size() << " reached, all covered by Reach(" << formula.size() << ")\n";
    else std::cerr << "[REACHTEST FAIL] (" << K1 << "," << K2 << "): " << bad << " uncovered\n";
    return bad ? 1 : 0;
}
```

```cpp
uint64_t RetrogradeSolver::selective_preload_bytes(int K1, int K2) const {
    auto micro = [](int Mp) -> uint64_t {
        uint64_t bc = StateIndex::nCr(50 - Mp + 9, 9);
        return 2 * ((bc + 7) / 8);                 // win + draw planes, bit-packed
    };
    uint64_t total = 0;
    for (int j = K1 + 2; j <= 24; j += 2) total += micro(K2 + j);
    for (int j = K2 + 2; j <= 24; j += 2) total += micro(K1 + j);
    return total;
}

StoreMode RetrogradeSolver::mode_for_pair(int K1, int K2, uint64_t pair_states) const {
    const char* env = std::getenv("BESTEMSHE_STORE_MODE");
    if (env) {
        std::string e(env);
        if (e == "byte") return StoreMode::ONE_BYTE;
        if (e == "packed") return StoreMode::TWO_BIT;
    }
    uint64_t need = pair_states + selective_preload_bytes(K1, K2);   // 1-byte store + selective preload
    return (need <= RAM_BUDGET) ? StoreMode::ONE_BYTE : StoreMode::TWO_BIT;
}
```

```cpp
template <class Store>
void RetrogradeSolver::solve_pair_impl(Store& store, int K1, int K2, uint64_t b_count) {
    int slices = (K1 == K2) ? 1 : 2;
    int min_K = StateIndex::GetMinK(layer_M);
    constexpr uint64_t BLOCK = 65536;

    while (true) {
        std::atomic<uint64_t> proven{0};
        for (int sl = 0; sl < slices; ++sl) {
            int K_self = (sl == 0) ? K1 : K2;
            uint64_t I_K = (K_self - min_K) / 2;
            uint64_t slice_base = static_cast<uint64_t>(sl) * b_count;
            uint64_t num_blocks = (b_count + BLOCK - 1) / BLOCK;

            #pragma omp parallel for schedule(dynamic, 1)
            for (uint64_t blk = 0; blk < num_blocks; ++blk) {
                uint64_t jb = blk * BLOCK;
                uint64_t je = std::min(b_count, jb + BLOCK);
                SolverState s = UnindexState(I_K * b_count + jb, layer_M);   // sets K_self, K_opp, board
                for (uint64_t j = jb; j < je; ++j) {
                    uint64_t local = slice_base + j;
                    if (store.read(local) == GameValue::UNKNOWN) {
                        bool all_losses_for_me = true;
                        bool proved_win = false;
                        for (int move = 0; move < 5; ++move) {
                            if (s.board[move] == 0) continue;
                            SolverState next_s;
                            bool empties_opponent;
                            bool is_capture = execute_and_flip(s, move, next_s, empties_opponent);
                            GameValue tv;
                            if (empties_opponent || next_s.K_opp >= 26) tv = GameValue::LOSS;
                            else if (is_capture) tv = inference_engine->query_state(next_s.K_self + next_s.K_opp, next_s.K_opp, IndexState(next_s));
                            else {
                                int tslice = (next_s.K_self == K1) ? 0 : 1;
                                tv = store.read(static_cast<uint64_t>(tslice) * b_count + StateIndex::IndexBoard(next_s.board));
                            }
                            if (tv == GameValue::LOSS) { proved_win = true; break; }
                            if (tv == GameValue::DRAW || tv == GameValue::UNKNOWN) all_losses_for_me = false;
                        }
                        if (proved_win) { store.write(local, GameValue::WIN); proven.fetch_add(1, std::memory_order_relaxed); }
                        else if (all_losses_for_me) { store.write(local, GameValue::LOSS); proven.fetch_add(1, std::memory_order_relaxed); }
                    }
                    if (j + 1 < je) StateIndex::AdvanceBoard(s.board);
                }
            }
        }
        if (proven.load(std::memory_order_relaxed) == 0) break;
    }

    #pragma omp parallel for schedule(dynamic, 65536)
    for (uint64_t local = 0; local < static_cast<uint64_t>(slices) * b_count; ++local)
        if (store.read(local) == GameValue::UNKNOWN) store.write(local, GameValue::DRAW);
}

template <class Store>
void RetrogradeSolver::write_pair_raw(const Store& store, int K1, int K2, uint64_t b_count, int slices, const std::string& out_dir) {
    uint64_t b_bytes = (b_count + 7) / 8;
    for (int sl = 0; sl < slices; ++sl) {
        int k1 = (sl == 0) ? K1 : K2;
        int k2 = (sl == 0) ? K2 : K1;
        std::vector<uint8_t> win(b_bytes, 0);
        std::vector<uint8_t> draw(b_bytes, 0);
        uint64_t base = static_cast<uint64_t>(sl) * b_count;
        for (uint64_t j = 0; j < b_count; ++j) {
            uint8_t v = static_cast<uint8_t>(store.read(base + j));
            if (v == static_cast<uint8_t>(GameValue::WIN)) win[j / 8] |= (1 << (j % 8));
            else if (v == static_cast<uint8_t>(GameValue::DRAW)) draw[j / 8] |= (1 << (j % 8));
        }
        std::string stem = out_dir + "/layer_" + std::to_string(k1) + "_" + std::to_string(k2) + "_";
        std::ofstream(stem + "win.raw", std::ios::binary).write(reinterpret_cast<const char*>(win.data()), win.size());
        std::ofstream(stem + "draw.raw", std::ios::binary).write(reinterpret_cast<const char*>(draw.data()), draw.size());
    }
}

void RetrogradeSolver::solve_layer_pairwise(const std::string& out_dir) {
    std::filesystem::create_directories(out_dir);
    int min_K = StateIndex::GetMinK(layer_M);
    int R = 50 - layer_M;
    uint64_t b_count = StateIndex::nCr(R + 9, 9);

    for (int K1 = min_K; K1 <= layer_M / 2; K1 += 2) {
        int K2 = layer_M - K1;
        int slices = (K1 == K2) ? 1 : 2;
        uint64_t pair_states = static_cast<uint64_t>(slices) * b_count;

        inference_engine->clear_cache();                               // bound RAM to one pair
        inference_engine->preload_pair(static_cast<uint8_t>(K1), static_cast<uint8_t>(K2));

        StoreMode mode = mode_for_pair(K1, K2, pair_states);
        if (mode == StoreMode::ONE_BYTE) {
            ByteStore store(pair_states);
            solve_pair_impl(store, K1, K2, b_count);
            write_pair_raw(store, K1, K2, b_count, slices, out_dir);
        } else {
            PackedStore store(pair_states);
            solve_pair_impl(store, K1, K2, b_count);
            write_pair_raw(store, K1, K2, b_count, slices, out_dir);
        }
        std::cout << "[INFO] Solved pair (" << K1 << "," << K2 << ") mode=" << (mode == StoreMode::ONE_BYTE ? "1byte" : "2bit") << "\n";
    }
    std::cout << "[SUCCESS] Pairwise-solved layer M = " << static_cast<int>(layer_M) << "\n";
}
```
